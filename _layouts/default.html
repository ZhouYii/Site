391OS Operating System

An operating system written in C and x86 assembly. Winner of a semester-long OS design competition at University of Illinois at Urbana Champaign.

  • Partially ported for Raspberry Pi(ARMv7), including interrupt emulation and syscalls.
  • Adequate video buffer for three virtualized terminal windows and supports up to four nested shells.
  • Pre-emptive multitasking allows programs such as grep, cat and paint to run in different terminals.

Webcam Finger Detector

A computer vision module written in python for a rock-paper-scissors game.

  • Applies a gaussian blur, and converts the resulting RGB image to HSV color format.
  • Portions of the image outside of a HSV threshold are discarded.
  • We find the contours of the remaining image.
  • Convex hulls are calculated for the contours.
  • Hull defects with angles smaller than 90 degrees are counted as fingers.

A Rogue in Siebel - Nethack Clone

A dungeon crawling RPG game written in C++, modeled after a famous game called Nethack (which is turn is modeled after the grand-daddy of all dungeon crawlers, Rogue)

  • Dynamically generated maps using BST trees. Guarantees all rooms are connected.
  • Allows third-party defined monsters and AI modules to run alongside monster and NPC modules.
  • Implements game saves by serializing important structures and RNG seeds to disk.
